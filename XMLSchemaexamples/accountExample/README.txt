Sample schema for representing the information associated with a bank account

account_n.xsd		schema written with nested style (anonymous types, a single root element admitted)
account_d.xsd		schema written with DTD-like style (anonymous types, all elements admitted as root element)
account_t.xsd		schema written with named types and a single element admitted as root)
accountSMin.xml		xml document that references the schema account_d.xsd
