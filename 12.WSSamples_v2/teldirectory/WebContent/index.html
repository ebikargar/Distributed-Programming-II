Teldirectory RESTful Web Service

The teldirectory web service is a web service that allows clients to search a telephone directory initially stored in an xml file. At service deploy time the directory is loaded into main memory using two tree-map structures. Search operations use these structures.

The two main resources of the service are named phones and ids, representing respectively the collection of phone numbers and the collection of user ids. On these resources it is possible to perform get operations.
